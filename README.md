This is a Maya Scatter Tool for Artists. This tool is largely fit for 3D artists that have to quickly create scenes and/or proof of concepts for visual composition. Meant for use with pre-existing assets, this tool will uniformly duplicate and scatter assets throughout a selected plane on Maya, with plenty of artist control on how.

The main benefit of this tool is to create visually appealing scenes with little time investment and manual production. It is mainly intended for artists working in an industry, with deadlines and strict time limits to account for.

Main Libraries:
- Pyside 2/Pyside 6
- shiboken2/shiboken6 
- random
- math

Maya Features:
- Object selection
- Duplication
- Instancing
- Transform Editing
- Surface Normals
- Grouping
- Custom UI

Scattering Features:
- Whole mesh scattering, Selected-face scattering, Vertex-based scattering, Curve-guided scattering
- Random scale
- Random Rotation
- Tilt variation
- Minimum Spacing to prevent overlap
- Seed system for reproducible random results
- Bake system to convert procedural generation into normal objects
